# Verified StarDust Version 1 and Version 2 Catalogue Snapshot

## Included records
- StarDust Version 1: 70 numbered tag records (`1-1-001` to `1-1-070`)
- StarDust Version 2: 70 numbered tag records (`1-2-001` to `1-2-070`)

## Official sources
- StarDust Version 1: https://world.pokemonmezastar.com/sg/tag/28933/
- StarDust Version 2: https://world.pokemonmezastar.com/sg/tag/31437/

## Verified from the official pages
- printed tag code
- StarDust version
- Pokémon name
- official top-level grouping:
  - Superstar Pokémon
  - Star Pokémon
  - ★2~4 Pokémon
- selected featured moves/mechanics described by the official page

## Important limits
The official pages do not provide a complete public machine-stat dataset for every tag.
Therefore:
- `energy` and numeric stat bars remain **Home Arena preview values**, not official arcade values.
- the catalogue does not claim unknown moves or hidden battle calculations.
- types and Pokédex IDs are standard Pokémon metadata used for the app's type system and portrait lookup.
- official tag artwork is not imported into the app.

## Existing user data
The IndexedDB migration adds missing verified catalogue entries but does not overwrite:
- owned quantities
- favourites
- notes
- manual `Needs Match` tags
