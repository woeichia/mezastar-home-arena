# Changelog — Home Battle Profile and first-attacker Battle Wheel

## Added

- Runtime Home Battle Profile resolver for every selectable tag.
- Existing catalogue records remain unchanged.
- Stored battle-ready move power is preserved when available.
- Documented moves without battle power receive star-based Home Power.
- Tags without a documented move receive a clearly labelled generated Home Move.
- Status or missing categories resolve to Physical/Special using the stronger offensive stat.
- Tap Race or Fresh-advantage winner enters the Battle Wheel first.
- Eight execution segments: Miss, Weak Hit, Hit, Strong and Critical.
- Player taps to stop and lock the execution multiplier.
- Locked result is retained through the temporary round-advance path.

## Not included yet

- Attack charge and Rainbow timing
- Type effectiveness and damage
- HP and knockout handling
- Second attacker Battle Wheel

The temporary **Store Result** round-advance control will be replaced by real attack resolution.
