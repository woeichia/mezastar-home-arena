# Changelog — Unknown Tag Matching

- Added a Match Tag action for owned `Needs Match` records in My Collection.
- Added `MatchUnknownTagModal` to search verified catalogue records by name, printed code, serial, or version.
- Confirming a match removes the temporary unknown record and transfers its owned copy count to the verified catalogue record.
- Existing catalogue ownership is preserved and combined up to the app copy limit of 9.
