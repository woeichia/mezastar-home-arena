# Phase A — Catalogue to My Collection

## What this update adds
- A Catalogue tab using the existing binder-card visual language.
- Search by Pokémon name, fixture code, or serial.
- Rarity and element filters.
- Select multiple tags, choose how many physical copies you own, then add them to My Collection.
- My Collection shows only owned records by default.
- Existing saved ownership is kept and upgraded to include copy counts.

## Current limitation
The available data remains a small testing fixture list. It is not a verified official Mezastar catalogue and its displayed demo stats/energy values must not be treated as official arcade values.

## Next content stage
After this workflow is accepted, import one verified series/version with source fields, tag code, Pokémon name, category/rarity, documented move/features, and an import report.
